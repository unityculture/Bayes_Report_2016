CPC

Cost Per Click;Cost Per Thousand Click-Through 每點擊成本

  • 目前主流
  • 每點擊一次計費一次
    • 對於廣告主：提高了作弊的難度
    • 對於媒體：沒有點擊等於白放 -> 浪費廣告版位

CPC

廣告主在意的指標:

  • 流量
    • Impression：廣告的曝光量，多少使用者看到這個廣告。

  • 成效
    • Ex: CTR(點閱率) = 點擊廣告次數(Click)/曝光人數(Impression)

研究問題

參考文章

ROC & AUC

ROC Curve 為判斷預測模型效果的指標之一

其中透過AUC的值可以判斷預測模型好壞：

根據AUC一般定義：

  • AUC=0.5 (no discrimination 無鑑別力)
  • 0.7≦AUC≦0.8 (acceptable discrimination 可接受的鑑別力)
  • 0.8≦AUC≦0.9 (excellent discrimination 優良的鑑別力)
  • 0.9≦AUC≦1.0 (outstanding discrimination 極佳的鑑別力)

ACC

Accuracy Rate = 被模型正確分類的User個數 / User總數
視作整體被分類正確的比例。

但若是我們今天關心的只是，我們預測為女生且真的是女性的機率(如：想投放彩妝商品廣告給女性)，用Accuracy Rate會有問題

被模型正確分類的User = 被模型正確分類的男性User + 被模型正確分類的女性User
可能Accuracy Rate很高的情況下，但被模型正確分類的女性User為0。

因此，需要另外一個指標，Precision Rate。

Precision Rate vs. Recall Rate

[準確度] 與 [量] 的 Trade-off
是否要犧牲準確度換取更多的量？

  • Model Output為機率值 -> 門檻值(Cut Point) -> 分類
  • 門檻值設定的越高 -> 找出相對是女性的機率較高的一群 -> 數量少 -> Precision高
  • 門檻值設定的越低 -> 找出相對是女性的機率較不高的一群 -> 數量高 -> Precision低
